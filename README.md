# Personal-Finance-Tracker
Personal Finance Tracker is a Java console-based application developed using Object-Oriented Programming (OOP) concepts. The system allows users to manage their personal finances by adding income and expenses, checking current balance, viewing transaction history, and generating monthly financial reports.

The project uses separate classes for Account management, Transactions, Income, Expense, and Report generation to maintain clean structure and modularity. It demonstrates important OOP concepts such as Abstraction, Encapsulation, Inheritance, and Polymorphism.

Features:
• Create account with initial balance
• Add income transactions
• Add expense transactions
• Automatically update account balance
• Generate monthly financial report
• View complete transaction history
• Check current balance
• User-friendly menu-driven console interface

Technologies Used:
• Java
• OOP Concepts
• ArrayList Collection
• Scanner Class

OOP Concepts Implemented:
• Abstraction using abstract Transaction class
• Inheritance through Income and Expense classes
• Polymorphism using apply() method overriding
• Encapsulation using private data members and getter methods
